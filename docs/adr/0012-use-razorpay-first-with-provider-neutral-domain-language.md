# Use Razorpay First with Provider-Neutral Domain Language

TripOS will integrate Razorpay first for provider-confirmed payments in the India MVP, while keeping domain language provider-neutral through terms like **Provider Payment** and **Provider Payment Setup**. This gives the MVP one concrete payment path without baking Razorpay-specific naming into the product model, leaving room for future providers such as Cashfree or PhonePe.
