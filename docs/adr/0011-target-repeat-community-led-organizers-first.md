# Target Repeat Community-Led Organizers First

TripOS will target repeat **Community-Led Organizers** running paid trips with roughly 10-80 travelers per trip as the first beachhead. This excludes large travel agencies, corporate retreat planners, and one-off friend trips from the first design center, because repeat community-led organizers have acute WhatsApp and spreadsheet pain, meaningful payment volume, and operational workflows simple enough for a focused MVP to replace.
